# Smart Website Analyzer

## 🚀 Overview
A Java-based web scraper that extracts publicly available data from websites.

## 🔧 Tech Stack
- Java (Spring Boot)
- JSoup
- REST API
- HTML/CSS UI

## ✨ Features
- Extract company name
- Extract description
- Extract emails
- Extract phone numbers
- Extract social links
- Extract logo URL
- Handles missing data (returns "Not Available")

## 🎯 Input
User enters a website URL

## 📤 Output
Structured JSON response with extracted data

## ⚠️ Limitations
- Only homepage scraping
- No JavaScript rendering
- No deep crawling

## 📌 Example
Input: https://zoho.com  
Output: Company name, description, etc.
